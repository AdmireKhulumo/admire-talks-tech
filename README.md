# Admire Talks Tech

A personal blog where I write about AI engineering, coding patterns, and everything in between. Read it on [admiretalkstech.com](https://admiretalkstech.com/).

## Technical Details

Built with [Astro](https://astro.build/) and deployed on [Cloudflare Pages](https://pages.cloudflare.com/).

Blog posts are written in Markdown and live in `src/content/blog/`.

### Local Development

```bash
npm install
npm run dev      # localhost:4321
npm run build    # production build
npm run preview  # preview production build
```
